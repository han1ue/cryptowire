
# CryptoWire

CryptoWire is a modern crypto news dashboard built with React, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:
	```sh
	npm install
	```
2. Start the development server:
	```sh
	npm run dev
	```

## Features
- Real-time crypto news
- Saved articles
- Category filtering
- Responsive terminal-inspired UI
